# Introduction
We build our project to help people to reach and git their favorites animal 
with good price , also help people who wants to sell their animal by putting information about their animal in our website
and help other people to see and reach to it.

# Project Features
at the beggining our website have two levels of user (Admin  , Normal User)
Normal user features , every user can create his own account and login to see all animals in the website ,
also user can add his animal to sell it by providing information and image to another user and send request to admin to accept his addition or not , also each user can make a search accourding to animal name 
, Finally we divide animals in categories to help people to choose from specific category


for the admin , he can do all the features mentioned for normal user ,  in addition to create new category , edit and delete for each animal in website ,after user add new animal at the first this animal send as a request to admin to accept or reject it
to make some restriction in website and prevent garpish data
